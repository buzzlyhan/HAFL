# HAFL Tools for VS Code

`.hafl` 向けの最小構成 VS Code 拡張です。  
HAFL Strict v0.3 を前提に、基本的な検証と補完を行います。

## 含まれる機能

- `.hafl` の言語登録
- コメント、セクション、キーの簡易ハイライト
- セクション、キー、値、ブロックを分けた強めの色分け
- 入力中のバリデーション診断
- セクション名と推奨キーの補完
- `true` `false` `null` の値補完
- `@begin ... @end` ブロックスニペット
- アウトライン表示
- セクションとブロックの折りたたみ
- `HAFL: Go to Section` コマンド

## 現在のバリデーション対象

- セクション外の内容
- 同一セクション内の重複キー
- `@begin` と `@end` の不整合
- `@begins` の誤記
- 無効なセクション名、キー名
- 標準外セクションや曖昧キーの警告

## 使い方

1. このフォルダを VS Code で開く
2. `F5` で Extension Development Host を起動する
3. 開いた側で `.hafl` ファイルを編集する

コマンドパレットから `HAFL: Go to Section` を実行すると、文書内のセクションへジャンプできます。
